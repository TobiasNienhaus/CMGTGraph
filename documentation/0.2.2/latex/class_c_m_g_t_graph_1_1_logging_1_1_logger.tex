When you get it, you get the current text writer of this logger. 

When you set it, the default text writer will be saved in \mbox{\hyperlink{class_c_m_g_t_graph_1_1_logging_1_1_logger_a4c51cb2f78ad0beb21bcdae211d2b36b}{\+\_\+default\+Out}} (to be able to reset it) and the out will be set. If the provided value is null, nothing happens. 

In the current implementation, \mbox{\hyperlink{class_c_m_g_t_graph_1_1_logging_1_1_logger_ad49636614eaa9eed7aaa6ff13064ddbb}{Writer}} is always equal to Console.\+Out, as it simply sets the out of the console.
